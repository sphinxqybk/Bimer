# Timer App - แอปนาฬิกาจับเวลา

แอปนาฬิกาจับเวลาที่มีฟีเจอร์ครบครัน รองรับทั้งการจับเวลาและนับถอยหลัง

## ฟีเจอร์

### ⏱️ จับเวลา (Stopwatch)
- เริ่ม/หยุด/รีเซ็ตเวลา
- จับรอบ (Lap timing)
- แสดงเวลาถึงหลักร้อยของวินาที
- แสดงรายการรอบที่จับได้

### ⏰ นับถอยหลัง (Countdown Timer)
- ตั้งเวลาได้ในหน่วยชั่วโมง นาที วินาที
- Progress bar แสดงความคืบหน้า
- เตือนเมื่อเหลือเวลา 10 วินาที
- เสียงแจ้งเตือนเมื่อหมดเวลา
- การแจ้งเตือนผ่าน browser notification

## การใช้งาน

### เปิดแอป
1. เปิดไฟล์ `index.html` ในเว็บเบราเซอร์
2. เลือกแท็บที่ต้องการใช้งาน

### จับเวลา
1. คลิก "เริ่ม" เพื่อเริ่มจับเวลา
2. คลิก "หยุด" เพื่อหยุดชั่วคราว
3. คลิก "จับรอบ" เพื่อบันทึกเวลาปัจจุบัน
4. คลิก "รีเซ็ต" เพื่อเริ่มใหม่

### นับถอยหลัง
1. ตั้งเวลาในช่องชั่วโมง นาที วินาที
2. คลิก "เริ่มนับถอยหลัง"
3. คลิก "หยุด" เพื่อหยุดชั่วคราว
4. คลิก "รีเซ็ต" เพื่อเริ่มใหม่

## คีย์บอร์ดช็อตคัต

- **Space**: เริ่ม/หยุด timer
- **R**: รีเซ็ต timer
- **L**: จับรอบ (เฉพาะโหมดจับเวลา)
- **1**: เปลี่ยนไปแท็บจับเวลา
- **2**: เปลี่ยนไปแท็บนับถอยหลัง

## เทคโนโลยีที่ใช้

- **HTML5**: โครงสร้างหน้าเว็บ
- **CSS3**: การออกแบบและ animations
- **JavaScript**: ฟังก์ชันการทำงานและ timer logic
- **Web Audio API**: เสียงแจ้งเตือน
- **Notification API**: การแจ้งเตือนของเบราเซอร์

## ฟีเจอร์พิเศษ

- **Responsive Design**: รองรับการใช้งานบนมือถือ
- **Visual Feedback**: เอฟเฟกต์แสงและสีเมื่อใช้งาน
- **Progress Visualization**: แถบความคืบหน้าสำหรับ countdown
- **Lap Tracking**: บันทึกและแสดงรอบที่จับได้
- **Audio Alerts**: เสียงแจ้งเตือนเมื่อหมดเวลา
- **Keyboard Support**: รองรับการใช้งานด้วยคีย์บอร์ด

## การติดตั้งและรัน

1. ดาวน์โหลดไฟล์ทั้งหมด
2. เปิดไฟล์ `index.html` ในเว็บเบราเซอร์
3. ไม่ต้องติดตั้งอะไรเพิ่มเติม รันได้ทันที!
